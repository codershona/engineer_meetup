# engineer_meetup

This is an Event Management Frontend Application 

## Technologies Used

* Vuetify 2.4.0
* Vue 0.1.0
* Firebase

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
